## Reflection

**What was the most surprising thing you learned today?**

The flex-wrap behavior. My horizontal pills ended up looking vertical because each item was wider than half the column, so `flex-wrap: wrap` pushed each one onto its own line. I couldn't tell whether the flex was actually working until I dragged the browser wider in DevTools — at a wider width they snapped into a horizontal row. That made the wrapping logic click.
